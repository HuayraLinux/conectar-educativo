Acerca de escritorio-educativo

Esta aplicación ha sido desarrolla en conjunto por el equipo técnico de educ.ar y huayra-linux. El desarollo comenzó como un mini-proyecto dentro hackatón del primer foro regional de program.ar.

El proyecto está basado en tecnologías HTML5, junto a varias herramientas de software libre, como nodewebkit, angularjs y bootstrap. Incluso las herramientas de desarrollo se pueden abrir para inspeccionar y modificar la aplicación.

El código completo del proyecto se puede acceder desde github, y es software libre.
